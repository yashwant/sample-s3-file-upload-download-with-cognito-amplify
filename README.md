#Sample App
This is just a sample app to get/store files to S3 Bucket.
with cognito authentication

##Steps
- Clone to your machine
```
$ git clone <repo url>
```

- Install dependencies
```
$ npm install
```

- Rename config.js.tp to config.js in src folder
```
$ cd src

$ mv config.js.tp config.js
```

- Run the app
```
$ npm run

```
